# Blog Application

A minimal React blog application with localStorage, dark/light mode, and multi-language support.

## Features

- Create, edit, and delete blog posts
- Data persists in browser localStorage
- Dark/Light mode toggle
- Multi-language support (English, Hindi, Spanish, French, German, Tamil, Telugu, Bengali, Marathi, Gujarati)
- Responsive design

## Installation

```bash
cd blog-app
npm install
```

## Run

```bash
npm start
```

The app will open at http://localhost:3000

## Usage

1. Click "New Post" to create a blog post
2. Enter title and content, then click "Save"
3. Use "Edit" or "Delete" buttons on each post
4. Switch languages using the dropdown
5. Toggle dark/light mode with the theme button

All data is stored in your browser's localStorage.
